## What is Denari?

Denari is a powerful, open-source accounting platform built for growing businesses. It gives you complete control over your finances — from invoicing and bank feeds to real-time reporting and multi-currency support — all in one place, on infrastructure you own.

## Features

<table>
  <tr>
    <td>📄 <b>Invoicing & Billing</b><br/>Create professional invoices, estimates, and credit notes. Track payments and send reminders.</td>
    <td>🏦 <b>Bank Feeds</b><br/>Connect your bank accounts for live transaction feeds with smart auto-categorization.</td>
  </tr>
  <tr>
    <td>📊 <b>Financial Reports</b><br/>Real-time profit &amp; loss, balance sheets, cash flow statements, and trial balances.</td>
    <td>🌐 <b>Multi-Currency</b><br/>Full support for international transactions and currency conversion.</td>
  </tr>
  <tr>
    <td>🏢 <b>Multi-Tenant</b><br/>Manage multiple organizations from a single installation.</td>
    <td>📦 <b>Inventory Management</b><br/>Track stock levels, costs, and product movements.</td>
  </tr>
  <tr>
    <td>📒 <b>Double-Entry Accounting</b><br/>A complete, standards-compliant general ledger at the core.</td>
    <td>🔓 <b>100% Open Source</b><br/>Full source code access with zero licensing fees.</td>
  </tr>
</table>

## Getting Started

```bash
git clone https://github.com/Technology-Wizard/Denari.git
cd Denari
docker compose up -d
```

Then open [http://localhost:8088](http://localhost:8088) in your browser.
